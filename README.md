- `ROS2数据采集脚本`: 采集配置文件话题列表ROS2BAG包，可以分析话题是否存在
- `autoware按钮遥控`: 通过点击按钮实现autowre的engage和速度下发
- `autoware启动按钮`: 通过点击按钮实现autowre的稳定启动和稳定关闭
